# Numbers That Matter — Owner Dashboard Guide

> Franchise Binder §7 · drafted 2026-07-22 · reader: the owner, 10 minutes a week, no dashboards opened. Everything below arrives as texts via the existing brief pipeline; this page explains what each number means and what "bad" looks like.

## Daily text (7am — yesterday's scorecard)
| Number | Healthy | Bad looks like |
|---|---|---|
| Rides completed / late / no-show | all completed, 0 no-show | any no-show without a logged fire |
| Calls the dispatcher had to make | 0–2 | >5 = a chronic driver problem, check tiers |
| Tomorrow's unassigned rides | 0 | any — cover today, not tomorrow 5am |
| Open fires | none | any fire open >24h |

## Weekly text (Sunday night)
| Number | Why it matters | Bad looks like |
|---|---|---|
| Margin $ and % | the business, in one line | school-year margin % drifting down 2+ weeks in a row |
| On-time acceptance rate | leading indicator of every other problem | <95% — look at the chronic tier list |
| Fires by category | is one category repeating? | same category 3 weeks running = playbook gap |
| Drivers gained / lost | churn signal | losing trusted-tier drivers (exit-interview them) |
| Rides offered vs accepted | growth signal — are we leaving money on the table? | passing on profitable rides for lack of drivers = hire |
| Estimated operator-hours | THE franchise metric | trending up = something regressed to manual |
| Paychex match | must say "exact" | anything else — that's a Monday incident, not a stat |

## Monthly text
- **Revenue + margin trend** — direction over 3 months beats any single week.
- **Driver count trend + revenue per driver** — growth quality, not just growth.
- **% of rides with a standing backup** — coverage resilience; target climbs toward 100%.
- **Extraction progress** — % of drivers with home area + reliability tier on record. When this is high, the business no longer lives in anyone's head.
- **Compliance horizon** — items expiring in the next 60 days, already nagged.

## The three questions the numbers answer
1. **Did the week make money?** Margin line, Paychex exact-match, reconciliation clean (no dispute windows burning).
2. **Does the machine run without the family?** Operator-hours + calls-made + % self-serve onboarding.
3. **Can it grow?** Offered-vs-accepted + backup coverage + driver pipeline.

## Ride-worth: take or pass (the standing decision rule)
Every new ride offer is scored on the four questions the intake tool computes and shows: **where is it · who's near it · what does it pay · how easy to cover.** The margin banner is the floor: red (thin/negative margin) = pass unless it's strategic (wheelchair/district-relationship rides are deliberately pass-through — that's a relationship investment, not a mistake). Each take/pass logs a one-tap reason; after ~10 decisions the pattern becomes an explicit written rule.
